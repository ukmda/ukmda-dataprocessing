Daily Simplified Report of Detections

This page provides a list of all detections in the last 72 hours from the UKMON network, primarily for the purposes of data exchange with other networks. Minimal camera details are also available with sufficient precision to estimate if there could be a field of view overlap.

If you've found an interesting match with another network please contact us via ukmeteornetwork@gmail.com so that we can exchange data with you.

The data here are released under the CC BY 4.0 license, so if you are using the data whether for scientific or other purposes, your must reference this web site and UKMON in your work.
